# Francisco Manneh — Portfolio

Live site: https://franciscomanneh.com

This repository contains a static personal portfolio built with HTML, CSS, and a small amount of JavaScript.

Quick notes
- Deploys from the `main` branch to GitHub Pages.
- Custom domain: `franciscomanneh.com` (CNAME included).

How to work locally
1. Clone the repo:

   git clone https://github.com/Franciscomanneh/FranciscoManneh.com.git

2. Open `index.html` in a browser or run a simple local server:

   # Python 3
   python -m http.server 8000

Pages / GitHub Actions
- A basic Pages workflow is included at `.github/workflows/pages.yml` to deploy on pushes to `main`.

Security
- If you created any tokens to enable Pages settings, revoke them when finished.
